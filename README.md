## 项目背景
### 电商模式
市面上有5种常见的电商模式B2B,B2C,C2B,C2C,O2O;
1、B2B模式
是指商家与商家建立的商业关系。如：阿里巴巴
2、B2C模式
就是我们经常看到的供应商直接把商品卖给用户，即商家对客户模式，也就是通常说的商业零售，直接面向消费者销售产品和服务。
如：苏宁易购，京东，天猫，小米商城
